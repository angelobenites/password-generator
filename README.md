# PassKey

Generador de Contraseñas, elaborado con Python, utilizando librerías para mejorar la seguridad de sus claves. **Recuerda que al generarlas debe copiarlas y guardarlas en un lugar seguro**.

## La contraseña contendrá: 
- Letras (Mayúsculas o Minúsculas)
- Símbolos
- Números

---

La aplicación se usa en la terminal, si usted desea abrirla con un comando, puede mover el archivo de la app: `passkey.exe`, en la ruta `C:\MisHerramientas`, después dirigirte a _Variables de Entorno_, en `Path`, agrega la ruta en donde haz puesto el archivo exe, ahora en la terminal ejecuta `passkey` y listo!
