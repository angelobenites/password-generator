# PassKey

Generador de Contraseñas, elaborado con Python, utilizando librerías para mejorar la seguridad de sus claves. **Recuerda que al generarlas debe copiarlas y guardarlas en un lugar seguro**.

## La contraseña contendrá: 
- Letras (Mayúsculas o Minúsculas)
- Símbolos
- Números

---

## Instalación

El mejor paso es usarla, así que vamos ayudarte con la instalación, lo primero que debes hacer el [ir a Lanzamientos](https://github.com/angelobenites/password-generator/releases), allí presiona el más reciente (osea el primero), después haz click en el archivo `passkey.exe`, este es la aplicación para Windows, que debe ser mayor a W10. 

Al presionar se descargará, después **ábrelo como administrador**, se abrirá la terminal, y deberás colocar la longitud deseada para la contraseña (recomendamos: 30).

![App Abierta](/assets/open-app.png)

Pero, nosotros lo hacemos mejor, con el archivo ya descargado. Crea tu carpeta dentro de `C:\`, supongamos que se llame `angelo.tools`, entonces muéve tu `passkey.exe` a `C:\angelo.tools\`, ahora entra a *Cambiar Variables de Entorno* (propio de windows), presiona editar, con esto entra a `Path`, añade la siguiente la ruta en donde se encuentra la app: `C:\angelo.tools\passkey.exe`.
